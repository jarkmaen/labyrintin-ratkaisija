# Ohjelman testaus

## Yksikkötestit

Yksikkötestit suoritetaan komennolla

```
mvn test
```

Testikattavuusraportti luodaan komennolla

```
mvn jacoco:report
```

Raportin tulos löytyy _target/site/jacoco/_ -hakemistosta nimellä _index.html_
